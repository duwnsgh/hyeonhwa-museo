# 墨夜 — 동양화로 그린 벗

현화고등학교 2학년 6반 미술 프로젝트로 만든 온라인 전시입니다.
트럼프·화투·타로 카드를 동양화로 다시 그리고, 그 안에 반 친구들의 모습을 담았습니다.
어두운 배경 위를 가로로 넘기며 한 점씩 감상하는 두루마리 형식의 갤러리예요.

## 보기

- `index.html`을 브라우저로 열면 바로 볼 수 있습니다.
- 좌우 화살표(← →), 마우스 휠, 스페이스바로 작품을 넘깁니다.
- 오른쪽 위 버튼으로 배경음악을 켜고 끌 수 있어요.

## 작품 추가 / 수정

작품 목록은 `index.html` 안의 `ARTWORKS` 배열에 모여 있습니다.

1. 그림을 `assets/img/` 폴더에 넣고 (예: `assets/img/타로카드-연인.png`)
2. 배열에 항목을 한 줄 추가하면 끝입니다.

```js
{
  img: '타로카드-연인',          // 확장자 빼고 파일명만
  title: '벗 其十一',
  subtitle: 'The Eleventh Companion',
  desc: '첫 줄.\n둘째 줄.\n셋째 줄.'   // 줄바꿈은 \n
}
```

낙관(도장)에 들어가는 한자 숫자는 순서대로 자동으로 붙습니다.

## 구성

- `index.html` — 전시 페이지 전체 (HTML·CSS·JS 한 파일)
- `assets/img/` — 작품 이미지
- `assets/music/` — 배경음악

별도의 빌드나 설치 과정은 없습니다. 순수 HTML/CSS/JavaScript로 되어 있어요.
